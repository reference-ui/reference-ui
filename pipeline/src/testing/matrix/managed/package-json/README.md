# Managed Matrix Package JSON

This folder owns every package.json shape generated by the matrix pipeline.

There are two related but distinct outputs:

- managed fixture package.json files inside the monorepo `matrix/*` folders
- synthetic consumer package.json files generated inside Dagger containers

Keeping both shapes here avoids spreading package.json rules across matrix sync code and runtime orchestration.

## Why This Exists

The matrix pipeline needs one source of truth for:

- which dependency versions are managed centrally
- which scripts route back through pipeline
- which raw consumer scripts still run inside the isolated Dagger sandbox
- how workspace protocol dependencies are rewritten to staged registry versions
