# Get Started

## Installation

```bash
pnpm add @reference-ui/react
```

## Usage

Import the components and styles you need:

```tsx
import { Button, Div } from '@reference-ui/react'
import '@reference-ui/react/styles.css'

function App() {
  return (
    <Div>
      <Button>Click me</Button>
    </Div>
  )
}
```
