# Light & Dark Mode

This page is a simple proof that theme-aware tokens are working in the docs app.

The demo below uses docs-local tokens defined with a default value and a `dark` override. Toggling the preview changes the `data-panda-theme` scope, and the same token names resolve to different colors without swapping components or styles.
