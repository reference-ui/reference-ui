# Icons

This page smoke-tests the packaged `@reference-ui/icons` artifact consumed by `reference-docs`.

<P fontSize="sm" color="docsMuted" marginTop="0" marginBottom="4r">
  `reference-docs` installs the local `reference-icons` package through a `file:` dependency at the
  package root, not a `workspace:*` dependency. That means docs still resolves the package through
  its package metadata and built `dist` entrypoints, while keeping the setup simple.
</P>

## Tokens, background, and `size`

Material icons render with `fill="currentColor"`, so **`color` on a parent** (for example `green.500`) controls the glyph. **`size`** maps to **`width` and `height`** on the icon’s outer **`Div`**, so tokens (`5r`), **`rem`**, numbers, **`calc()`**, etc. behave like everywhere else. This app’s **`ui.config.ts` `include`** also lists **`../reference-icons/src`** so **`ref sync`** runs Panda on `@reference-ui/icons` sources (not only `dist`), and those dimensions extract correctly.
