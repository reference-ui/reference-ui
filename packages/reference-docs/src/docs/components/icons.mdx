# Icons

Reference UI ships **Material Symbols** as React components (outline and filled, weight 500), generated from [`@material-symbols-svg/react`](https://www.npmjs.com/package/@material-symbols-svg/react). Import icons from `@reference-ui/lib/icons` alongside the rest of the design system.

```tsx
import { SearchIcon, HomeIcon } from '@reference-ui/lib/icons'

export function Example() {
  return (
    <>
      <SearchIcon size={24} variant="outline" />
      <HomeIcon size={24} variant="filled" />
    </>
  )
}
```

Use the search field to find a glyph by component name. Toggle **Outline** and **Filled** to preview both variants.
