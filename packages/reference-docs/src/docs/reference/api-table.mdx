# Reference API Table

The `Reference` component renders API documentation from the generated Tasty manifest at runtime. For interfaces, that means a live members table without hand-maintained docs.

## Define a public type

```ts
/**
 * Public button props used to exercise the live reference table.
 */
export interface DocsReferenceButtonProps {
  /** Visible text shown inside the button. */
  label: string

  /** Disables pointer and keyboard interaction. */
  disabled?: boolean

  /**
   * Visual treatment for the button shell.
   *
   * @default "solid"
   */
  variant?: DocsReferenceButtonVariant

  /**
   * Called when the button is pressed.
   *
   * @param event - Pointer event metadata from the trigger interaction.
   * @param state - Snapshot of the button state at click time.
   */
  onPress?: (event: DocsReferencePressEvent, state: DocsReferenceButtonState) => void
}
```

Any exported symbol inside the docs app can be picked up by `ref sync` and exposed through `@reference-ui/types`.

## Render the table

```tsx
import { Reference } from '@reference-ui/types'

<Reference name="DocsReferenceButtonProps" />
```
