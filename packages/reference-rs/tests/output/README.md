# TypeScript scanner test output

Emitted bundles and metrics are written here when you run the TypeScript scanner tests. Structure mirrors the input: one folder per scenario.

- **`{scenario}/bundle.js`** – emitted bundle for that scenario (e.g. `generics/bundle.js`, `external_libs/bundle.js`).
- **`{scenario}/bundle-metrics.txt`** – timing/metrics for that scenario’s run.

See `../input/README.md` for the scenario-folder convention.
